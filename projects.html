  • Flux Bin

    • Flux Bin is a waste disposal bin to use in factories. It differentiates the metals from plastics using a buffer and a LED bulb. If there's a metal object passes through this bin, buffer will 'BEEP' and the LED will switch on to indicate that. Project was mainly built using electromagnetic concepts.

    This was designed as a group project in the 1st semester and it selected as the best innovative project among the whole batch and got a chance to represent it in the 'INNOVATE 2019' which was organized by University of Sri Jayawardenapura for its 60th anniversary, which held in BMICH Sri lanka.

  • Galaxian Game

    • Galaxian is a group project which was done as the project for the programming module in first year second semester(2019/2020). It is a console based game, designed with simple C++ implementations with 3 levels(easy, medium, hard). It has a simple UI(console based) with sounds and clip arts designed with symbols.

  • UNO Card Game

    • This is a UNO card game mainly designed with Object Oriented Concept using java as a group project(2020). It consists with an attractive graphical user interface as shown in the figure.

    The game has 4 options which are default(vs PC) player option and 3 multiplayer options(2 players, 3 players, 4 players). It has a GUI menu option to select the mode, instructions, about game option with sounds. Game will automatically play until a player score 500 than the others and It has basic instruction which are simillar to the original game.

  • BLOOM(A software designed to improve mental health)

    • This is a web application which is a software to improve the mental health of person(stress releasing). This an ongoing group project and It has three main options a chatbot, emotion detection with a music player according to the mood and stress releasing games.

    Chatbot and Emotion tracker is designed with machine learning concepts and all stress releasing games are designed with HTML, CSS and JS.
